 Image Gallery with Filter & Search

A responsive image gallery built using HTML, CSS, and JavaScript.

 Features
- Image grid with title and category
- Category filter using dropdown
- Case-insensitive search by image title
- Combined filter and search logic
- Responsive layout for all devices
- Lightbox image preview
- Smooth UI transitions and animations

 Tech Stack
- HTML5
- CSS3 (Grid & Animations)
- JavaScript (Vanilla)

 How It Works
Images are stored in a JavaScript array.  
Filtering and searching are done using array `filter()` methods without page reload.
